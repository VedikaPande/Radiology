# Medvision – Intelligent Radiology Learning Platform

Medvision is an interactive radiology learning platform designed to help medical students and radiology residents improve diagnostic skills through case-based practice, AI-assisted analysis, and structured study tools.

The platform provides a simulated radiology learning environment where users can explore real-world imaging cases, answer diagnostic questions, review flashcards, track learning progress, and organize study notes.

## Features

* **Radiology Case Library** – Browse curated CT, MRI, X-ray, and ultrasound cases with diagnostic explanations.
* **AI Image Analysis** – Upload medical images and receive simulated AI-assisted predictions and confidence scores.
* **MCQ Practice System** – Test knowledge with radiology-focused multiple-choice questions across different modalities.
* **Flashcards** – Rapid review of key radiology concepts and imaging patterns.
* **Study Notes** – Create and organize personal radiology notes.
* **Learning Analytics** – Track accuracy, study streaks, and weekly learning activity.
* **Notifications** – Receive updates on new cases, achievements, and study reminders.
* **User Profiles & Settings** – Manage account details, preferences, and security.

## Tech Stack

**Frontend**

* React / Next.js
* Modern medical-style UI dashboard

**Backend**

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT Authentication

## Purpose

Medvision aims to replicate the experience of radiology spotter exams and clinical case discussions in a digital learning environment, helping users strengthen pattern recognition and diagnostic reasoning.
